# 🔐 OTP-generator

Google Authenticator みたいな　ブラウザ完結型のワンタイムパスワード（TOTP）生成ツールです。

## 🌟 主な機能
- **安全な設計**: Web Crypto API を利用し、すべての計算をブラウザ内で行います。秘密鍵がネットワークに流れることはありません。
- **クイックコピー**: OTP をクリックするだけでクリップボードにコピー。
- **タイムバー**: 次のコード更新までの残り時間を視覚的に表示。
- **モバイル対応**: レスポンシブデザインで、外出先のスマホからも利用可能。

## 🛠 使い方
1. 本リポジトリの `index.html` をブラウザで開く。
2. サービス側で発行された **Base32形式の秘密鍵** を入力欄にペースト。
3. 生成された 6 桁のコードをログインに使用。

## 📂 ファイル構成
- `index.html` : UI、スタイル、および TOTP 生成ロジックのすべてが含まれています。

## ⚙️ 技術仕様
- **アルゴリズム**: TOTP (Time-based One-Time Password)
- **準拠規格**: RFC 6238
- **ハッシュ関数**: HMAC-SHA1
- **開発言語**: HTML5, CSS3, JavaScript (Vanilla JS)
